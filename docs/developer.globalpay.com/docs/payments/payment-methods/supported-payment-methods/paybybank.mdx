## Additional Fields Syntax

**Type** indicates whether the element is Mandatory (**M**), Optional (**O**), or Conditional (**C**) - dependent on another field.

| Name           | Type | Format     | Length | Description                                                                                                                             |
| -------------- | ---- | ---------- | ------ | --------------------------------------------------------------------------------------------------------------------------------------- |
| App to App URL | O    | xsd:anyURI | 1-255  | Custom app URL for redirecting the customer back to the app that triggered the payment. Overrides MerchantRedirectURL for mobile flows. |
