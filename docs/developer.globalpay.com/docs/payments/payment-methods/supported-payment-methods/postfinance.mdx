## Additional Fields Syntax

**Type** indicates whether the element is Mandatory (**M**), Optional (**O**), or Conditional (**C**) - dependent on another field.

| Name        | Type            | Format       | Length | Description                                                                 |
| ----------- | --------------- | ------------ | ------ | --------------------------------------------------------------------------- |
| Mobile View | See Code Sample | \[a-zA-Z\]\* | 0-3    | Set to Yes to display Postfinance's mobile variant. Otherwise leave blank.  |
| Mobile View | O               | \[a-zA-Z\]\* | 0-3    | Set to Yes to display Postfinance's mobile variant. Otherwise, leave blank. |
