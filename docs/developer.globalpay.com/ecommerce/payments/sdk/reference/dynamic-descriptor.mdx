## Set a Descriptor

<Infobox>
Dynamic Descriptor is currently only supported by acquirers Global Payments North America and Allied Irish Banks. Both the issuer and the acquirer must support this functionality. Consult with your account manager or support agent before using.
</Infobox>

  If you’re using this functionality and no descriptor is flagged, we’ll use your merchant name as the default. The merchant name is set when you create or update your account with us.
