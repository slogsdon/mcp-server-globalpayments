## Requirements
* .NET Core 1.0+
* .NET Framework 4.6+
* Mono 4.6+

## Installation

Installing the SDK into your solution is usually done using NuGet.

To install via [NuGet Manager Console](https://docs.nuget.org/consume/package-manager-console):

### NuGet

<Code id="XML" title="Markdown" language="XML" active>
```
Install-Package GlobalPayments.Api -Version 1.7.14
```
</Code>

### GitHub

Download the the latest release of the SDK from [GitHub](https://github.com/globalpayments/dotnet-sdk).
