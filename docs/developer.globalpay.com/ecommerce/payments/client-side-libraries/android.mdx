## Requirements

* Android 4.4+
* Android SDK 19 or later

## Installation

### Gradle

### Hosted Payment Page
<Code id="XML" title="" language="XML" active>
```
compile 'com.realexpayments.hpp.sdk:rxp-hpp-android:2.0'
```
</Code>

### API
<Code id="XML" title="" language="XML" active>
```
compile 'com.realexpayments.remote.sdk:rxp-remote-android:2.0'
```
</Code>

### Maven
Add this dependency to your project's POM:
<Code id="XML" title="" language="XML" active>
```
<dependency>
  <groupId>com.realexpayments.hpp.sdk</groupId>
  <artifactId>rxp-hpp-android</artifactId>
  <version>2.0</version>
</dependency>
```
</Code>

### Manual

1) Download the the latest release of the SDK from [GitHub](https://github.com/globalpayments/rxp-android/releases).
2) Add module _rxp-hpp-android_ into your project to use the HPP SDK.
3) If you want to use the card data validation library, add the module _rxp-remote-android_ into your project.
