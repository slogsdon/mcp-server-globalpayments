<ReleaseNoteSection title="Verifications">
    <ReleaseNoteDetail type="added">
        We have added the functionality for merchants to retrieve additional BIN data on the Payment Method and Verification resources. For verifications, BIN data can be returned for both raw and tokenized cards. The fields returned are:

        * card.issuer
        * card.funding
        * card.country
        * card.currency

        Please connect with your Account Manager for additional details and see if you can avail of this feature.
    </ReleaseNoteDetail>
</ReleaseNoteSection>
