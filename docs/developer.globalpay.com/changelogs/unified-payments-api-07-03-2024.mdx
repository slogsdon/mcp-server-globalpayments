<ReleaseNoteSection title="Transactions">
    <ReleaseNoteDetail type="added">
        Added a currency_conversion object in the response returned in the GET List of Transactions request. This object contains the following fields:

        - payer_amount
        - payer_currency
        - margin_rate_percentage
        - exchange_rate
        - commission_percentage
        - exchange_rate_source
        - exchange_source_time

    </ReleaseNoteDetail>
