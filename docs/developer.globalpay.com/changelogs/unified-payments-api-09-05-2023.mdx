<ReleaseNoteSection title="Transactions">
    <ReleaseNoteDetail type="changed">
        * The field "brand_reference" no longer has the dependency on the "stored_credential" object.
        * Updated the error message when stored_credential object is sent incorrectly in the Create a Sale request. Updated error message is now:
            * "error_code": "INVALID_REQUEST_DATA"
            * "detailed_error_code": "40007"
            * "detailed_error_description": "Request expects the following conditionally mandatory fields initiator, model and sequence."
    </ReleaseNoteDetail>
</ReleaseNoteSection>

<ReleaseNoteSection title="Funds">
